## Lancer le projet
```bash
# 1. ouvrir un terminal de commande et se rendre dans le dossier du projet
$ cd monDossier

# 2. installer les dépendences du projet
$ npm install

# 3. Lancer le projet
$ npm run dev

# 4. ouvrir un navigateur avec l'adresse
http://localhost:3000/
```
